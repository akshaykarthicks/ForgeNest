# Habit Tracker API Documentation

Documentation site for the Habit Tracker API - a Django REST API for tracking daily habits, building streaks, and challenging friends.

## Development

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 to view the documentation.

## Project Structure

- `content/docs/` - Documentation pages in MDX format
- `src/app/(home)/` - Landing page
- `src/app/docs/` - Documentation layout
- `src/app/api/search/` - Search functionality

## Features

- Built with Next.js and Fumadocs
- MDX-based documentation
- Full-text search
- Responsive design
- Syntax highlighting for code examples
